# BarberFlow

**BarberFlow** é um sistema profissional para gerenciamento de barbearias, focado na simplicidade, rapidez e inteligência. Desenvolvido para atender desde barbeiros autônomos até grandes barbearias.

## 🚀 Funcionalidades
- **Agendamento Público Inteligente**: Bloqueio de horários passados e cruzamento de horários ocupados.
- **Fila de Atendimento**: Atualização em tempo real (painel para TV).
- **Dashboard**: Métricas, serviços mais vendidos e fluxo de clientes.
- **Gestão Financeira**: Lançamentos automáticos baseados nos serviços concluídos.
- **Relatórios**: Exportação em PDF e Excel.
- **Instalador SaaS**: Assistente de instalação configurável (em desenvolvimento).

## 💻 Tecnologias
- Laravel 11.x
- PHP 8.2+
- MySQL / SQLite
- Bootstrap 5.3
- JavaScript (Vanilla ES6)
- FullCalendar & SweetAlert2

## 🛠 Instalação
1. Clone o repositório
2. Rode `composer install`
3. Rode `npm install` e `npm run build`
4. Copie o `.env.example` para `.env`
5. Configure o banco e e-mail no `.env`
6. Rode `php artisan key:generate`
7. Rode `php artisan migrate --seed`
8. Acesse `http://localhost/`
