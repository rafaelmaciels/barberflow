# Contributing to BarberFlow

Please review this document before submitting any pull requests.

1. **Issues**: Use o board de issues para discutir grandes alterações.
2. **Commits**: Utilize o padrão Conventional Commits (`feat:`, `fix:`, `refactor:`, `docs:`, etc).
3. **Padrão de Código**: Obedeça a PSR-12 e certifique-se de que nenhum erro é lançado antes do commit.
